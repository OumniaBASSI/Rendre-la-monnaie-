 un programme qui donne un nombre minimum de billets et de pièces, selon une somme que l’utilisateur a saisi au clavier.
Exemple : Si l’utilisateur a saisi 348 DH le programme doit afficher :
Billets 200 DH : 1
Billets 100 DH : 1
Billets 50 DH : 0
Billets 20 DH : 2
Billets 10 DH : 0
Pièces 5 DH : 1
Pièces 2 DH : 1
Pièces 1 DH : 1
